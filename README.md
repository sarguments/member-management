# member-management
간단한 멤버관리

## URI

| 메소드 | 경로                      | 설명      |
| ------ | ------------------------- | --------- |
| POST   | /members                | 사용자등록  |
| POST   | /members					 | 목록조회    |
| GET    | /members/{id}           | 상세조회    |

- OS - Ubuntu 18.04.1 LTS
- IDE - IntelliJ
- Language - Java 8
- Framework - Spring boot 2.0.6
